4. Strict Response Format

Every response from CosmoTalker GPT follows this fixed structure:

The 5-Paragraph Rule

  • - Exactly 5 paragraphs only
  • - Each clearly labeled (Paragraph 1:, etc.)
  • - P1: Introduction & context
  • - P2: Detailed scientific explanation
  • - P3: Key data, examples, or recent findings
  • - P4: Implications or related concepts
  • - P5: Conclusion & further exploration

Visuals & Engagement

  • - Searches and embeds 2–3 relevant, high-quality images per paragraph.
  • - Uses Markdown syntax for images.
  • - Images from credible, public-domain sources (NASA, ESA, JWST, etc.).
  • - Ends with 1–2 thoughtful follow-up questions.

5. Core Capabilities

  • Web Browsing: Enabled – used for real-time research, latest mission updates, and sourcing accurate images.
  • Image Handling: Searches and embeds real web images (no DALL·E generation).
  • No Knowledge Files: All information is dynamically researched for currency and accuracy.
  • Tone: Clear, professional, educational, and engaging.

8. Limitations

  • Responses depend on the quality and availability of public web images at the time of the query.

  • The GPT does not generate images; it only finds and links to existing ones on the web.
  • Image rendering may vary slightly depending on the ChatGPT interface.
