🌌 How to Use the CosmoTalker Browser API

The CosmoTalker Browser API allows frontend developers to execute CosmoTalker Python commands directly in the browser via JavaScript, returning clean JSON responses. No backend server is required!

Step 1: Import the SDK

Include the cosmo-api.js file in the <head> of your HTML document.

<script src="https://bhuvanesh-m-dev.github.io/cosmotalker/api/cosmo-api.js"></script>

Note: This exposes a global JavaScript object named CosmoAPI.

Step 2: Write an Async Function

Because Pyodide (the Python WASM engine) takes a few seconds to load the first time it is called, all API methods are asynchronous. You must use async/await or .then() chains.

async function mySpaceApp() {
    console.log("Contacting Cosmos...");
    
    // Call the planet info API
    const result = await CosmoAPI.planetInfo("Mars");
    
    if (result.status === "success") {
        console.log("Data received:\n", result.data);
    } else {
        console.error("Error:", result.message);
    }
}

Step 3: Trigger the Code

You can bind this function to a button click, a form submission, or call it as soon as the webpage loads.

<button onclick="mySpaceApp()">Explore Mars</button>

📚 Available API Methods

  • CosmoAPI.getFunFact() - Returns a random space fun fact.
  • CosmoAPI.planetInfo("Jupiter") - Fetches statistics for a specific planet.
  • CosmoAPI.search("black hole") - Searches the CosmoTalker DB for an astronomical topic.
  • CosmoAPI.spacex() - Gets the latest SpaceX launch information.
  • CosmoAPI.celestrak() - Retrieves Celestrak data.
  • CosmoAPI.wiki("Apollo 11") - Fetches Wikipedia summaries for space topics.
  • CosmoAPI.execute("print('Hello Space')") - Advanced: Runs custom Python/CosmoTalker code.
